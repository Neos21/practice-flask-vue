# Practice Flask Vue

Python Flask と Vue.js を組み合わせたプロジェクトのサンプル。

```sh
# Front-end
$ cd ./frontend/
# Install dependencies
$ npm install
# Production Build : Create ./frontend/dist/ directory
$ npm run build

# Back-end
$ cd ./backend/
# Install dependencies
$ pipenv install --dev
# Start : http://localhost:5000/ (Refer to ./frontend/dist/ directory)
$ pipenv run start
```
